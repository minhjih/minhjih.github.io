Shell Sort

Shell sort는 n칸 씩 건너뛰면서 Sorting하고, Adaptive하게 점점 n을 줄이면서 sorting한다.

Properties

  1. gap sequence 에 따라 향상시킬 수 있는데 어떤 gap sequence를 써야하는지는 아직 정확하진 않음
  2. Adaptive한 방식이라서, 언제 끝날지는 모름.-> 더 빨리 끝날 수 있다는 기대가 있음.
  3. Unstable함

void ShellSort(int a[], int n) {
	int i, j, tmp, t;
	for (t = n / 2; t >= 1; t /= 2) { //gap이 t로 정의됨
	//내부 sorting과정 어디는 selection sort, bubble sort인데 여긴 약간 삽입 정렬 느낌
	//상관 없는듯
		for (i = t; i < n; i++) {// t부터 시작해서 점점 증가해나감
			for (j = i - t; j >= 0; j-=t) { //gap이전것들과 하나씩 비교
			
			    if(a[j+t] > a[j]){// 자리 선점 되면 이제 멈춰
			        break;
			    }else{// 아직 새로 들어온것의 자리가 불안전 자리 바꿔줘야 함.
			        tmp = a[j];
			        a[j] = a[j+t];
			        a[j+t] = tmp;
			    }
			}
		}
	}
}

Merge Sort

Devide and conquer와 굉장히 유사하다.

Procedure

  1. Divide th array into two parts
  2. Divide the array into two parts again
  3. break into single parts
  4. sort하면서 array합침.
  5. divided arrays합친다.

Merge sort는 반복적인 작업들은 Recursion으로 만들어서 재귀 함수로 만들 수 있다.

def MergeSort(arr[], l, r) // target element l~ r까지 중에서

If r > l
	//1. Find Middle point
	middle m = (l+r)/2;
	//2. Call MergeSort
	Call mergeSort(arr, l, m);
	Call mergeSort(arr, m, r);
	//3. 서로 각자 merge sorted된 어레이를 merge하며 sort한다.
	Call merge(arr, l, m, r);

l~m까지 m~r까지는 정렬이 되어있으니, Merge과정에서 k++해 가면서(arr[k]), l~m에서 가져올지, m~r에서 가져올지 결정하면 된다
